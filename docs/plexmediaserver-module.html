Modules

The following modules provide some very useful tools to help maintain your Plex Media Server/s.

Click or tap on the icons below to take you to individual pages for more information about each module.

Download module

Use this tool to download media from any shared servers where you have been granted download privileges.
You will also require an active Plex Pass in order to use this module.

ExportTools module

The ExportTools module allows you to export any or all of the information about your media into a .csv and/or .xlsx file.
To customise the data output by ExportTools, click on ExportTools, then Custom Levels (displayed under the ExportTools menu item), to create any Custom Levels you may wish to use to export your media information with.

See the Custom Levels page for more details on creating a Custom Level etc...

Plex Media Server module

This module is the landing page for several useful sub-modules. You may select each of the available sub-modules from the side menu on the left of the WebTools-NG window.

Plex Media Server module

The Plex.TV module allows you to identify anyone you have shared your Plex Media Server/s with.
