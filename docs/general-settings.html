General Settings

Main Menu

The General Settings page allows you to set various parameters that control the WebTools-NG program. The various settings are explained below.

What file log level to use (Change only if asked by the devs)
The default setting is info. You should change it to verbose or any other setting only when asked to do so.

Console log level (only used when developing)
The default for this setting is info.

Size of logfile, before rollover
You should only increase/decrease this setting if asked to do so, or if trying to track down an elusive issue.

Subscribe to Beta updates
Enable this setting only if you wish to install Beta updates.

Subscribe to updates (if false, beta updates will also be disabled)
This will enable all regular updates, but if you disable it, it will also disable Beta updates.
