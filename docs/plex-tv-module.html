Plex.TV

Main Menu

The Plex.TV module allows you to identify anyone you have shared your Plex Media Server/s with.

Simply select the users name from the Select User drop down menu and you will be presented with the details of that user.

Note: * Click here for general settings regarding export *
Click on this link to take you to the Gloabl Settings > Export Settings page

Export user to CSV
Click on this button to save the users details to a .csv/.xlsx file

Export all users to CSV
Click on this button to save all users and their details to a .csv/.xlsx file.
