Beta Testing

Main Menu

Yes, we do have a Beta Testing program, and that's your chance to not only try out the bleeding edge of WebTools-NG, but also help us help the Plex community as a whole. Besides that, you as a Beta tester also engage directly with the developers of WebTools-NG, so if something is on your mind regarding WebTools-NG, then this is indeed the place to be.

If you want to participate in the beta program for WebTools-NG, then it does involve a few things:

Requirements

- You must accept, that this is beta software!
- You must sign up to our Slack account. Click on this Join Slack link and sign up.
- You must also click this GitHub link and create an account, if you do not already have one.
- You must also sign up for our translation site. Click on this POEditor link and sign up.
- Note that this is regardless of whether or not you do any translations. We use this information for our Credits page.

You must download the current WebTools beta (Link is available in Slack).

Language

You should test WebTools in your native language.
If a translation is not correct, you should correct it at the translation site.
If your language is missing, you should create and translate it at the translation site.

Testing

You should test as many functions of WebTools-NG as possible.

Bug Reporting

If you are facing a bug, and it's not listed in GitHub Issues, then you must create a new [BUG] Issue report.

Feature Requests

If you have a feature you would like to see included in WebTools-NG, or you have any suggestions for enhancements to existing features and they are not listed in GitHub Issues, then please create a new [RFE] Issue report.

Questions

Start by checking the Wiki or this Website first please.
If you still have a question during beta testing, it should be asked in our Slack beta-v3 channel.

Credits

If you sign up for the beta, you will be mentioned in the About Us page of WebTools-NG, with the name you use for our translation site, unless you opt out of that.

We hope everyone enjoys WebTools-NG and we would like to thank all of those people who have contributed to making WebTools-NG a great tool.

Catch!

If you simply sign up, but neither participate in the beta-testing, nor translate, then you will be removed from the credit list after ~ 3 weeks, this is to ensure, that only people that participate get any credit!
Thank You!
